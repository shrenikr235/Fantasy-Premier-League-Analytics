
# FPL Analytics

The project is aimed at analyzing FPL data to rate and rank players, analyze match event data in real-time, compute team strengths and understand and predict player ratings using Apache Spark and its supporting libraries.

## Prerequisites

Before you begin, ensure you have met the following requirements:
* You have installed the latest version of Apache Spark and Hadoop HDFS

## Folder Structure
![alt text](https://github.com/shrenikr235/bdproject/blob/master/structure.png?raw=true)

## To execute, follow these steps:

1. Fork and clone the repository or download as ZIP and extract.
2. Open two terminals
3. Go to `bdproject/data/stream/` and run `python3 stream2.py`
4. In the 2nd terminal go to `bdproject/code/` and run `spark-submit main.py`
5. The report for the project is located in `bdproject/`
